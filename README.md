
# 💻 Termux Personalizado

Este repositório fornece uma personalização automática para o seu Termux, com banner estilizado, atalhos úteis e aparência mais bonita. Tudo é instalado com apenas **1 comando**.

---

## ⚙️ O que está incluído:

✅ Banner colorido com seu nome  
✅ Neofetch ao abrir o terminal  
✅ Atalhos personalizados (alias)  
✅ Tema simples e prático  
✅ Instalação automática sem esforço

---

## 🚀 Instalação automática

Abra o Termux e execute este único comando:

```bash
bash <(curl -s https://raw.githubusercontent.com/4ykoor/termuxpreguica/main/install.sh)
````

Pronto! Seu Termux será personalizado em segundos.

---

## 🛠️ Atalhos disponíveis (alias)

| Comando  | Ação                                  |
| -------- | ------------------------------------- |
| `update` | Atualiza todos os pacotes             |
| `limpar` | Limpa a tela                          |
| `sair`   | Encerra o Termux                      |
| `banner` | Exibe o banner manualmente (opcional) |

---

## ✏️ Personalize ainda mais

Você pode editar o arquivo `.bashrc` para mudar frases, atalhos e até mudar o estilo do banner.

```bash
nano ~/.bashrc
```

Depois, salve e execute:

```bash
source ~/.bashrc
```

---

## 📦 Requisitos

* Termux instalado (última versão)
* Internet para rodar o comando de instalação
